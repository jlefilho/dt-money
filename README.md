### Instalação
1. Clone o repositório

```bash
git clone https://github.com/jlefilho/dt-money.git
```

2. Acesse a pasta do projeto

```bash
cd dt-money
```

3. Instale as dependências necessárias

```bash
npm install
```

4. Execute o projeto

```bash
npm run dev
```
